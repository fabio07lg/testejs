passos para um novo projeto:
  1. criar repositorio no github
  2. clonar o repositorio usando o TortoiseGit (sincronizar local com o github)
  3. escrever o código da aplicação
  3.1 se a aplicacao usar npm executar npm init e npm install PACOTE
  4. executar a aplicação com node APP.js
